# Canadian Tech Job Market Analysis

## 📊 Overview
Interactive Power BI dashboard analyzing 50 Canadian tech job postings from April 2025. This project demonstrates data analysis, SQL querying, and business intelligence visualization skills.

## 🎯 Key Findings

### Salary Insights
- **Senior Level Average:** $120,000+
- **Mid Level Average:** $100,000
- **Entry Level Average:** $75,000
- **Salary Gap:** Senior positions pay 60% more than entry-level roles

### Geographic Trends
- **Highest Paying Province:** Ontario (average $92K)
- **Job Distribution:** Ontario leads with most opportunities
- **Regional Growth:** BC, Quebec, and Alberta also strong markets

### Industry Analysis
- **Top Industry:** Finance (12 job openings)
- **Close Second:** SaaS (10 openings)
- **Growing Sectors:** Tech, Consulting, Telecom

### Top Hiring Companies
- Shopify, Deloitte, Microsoft Canada, RBC, TD Bank

## 🛠️ Tools & Technologies Used
- **Database:** MySQL 8.0
- **Analysis:** SQL (complex queries, aggregations)
- **Visualization:** Power BI Desktop
- **Data Modeling:** Power BI (DAX, calculated columns)
- **Design:** UI/UX principles for dashboard layout

## 📈 Dashboard Features

### Interactive Filters
- **Filter by Province:** Alberta, BC, Ontario, Quebec
- **Filter by Experience Level:** Entry, Mid, Senior
- All charts update dynamically based on selections

### Visualizations Included
1. **Average Salary by Province** - Shows salary trends with experience level breakdown
2. **Top Companies Hiring** - Displays companies with most job openings
3. **Jobs by Industry** - Horizontal bar chart showing industry distribution
4. **Salary by Experience Level** - Clear comparison of entry vs mid vs senior roles
