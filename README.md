# Canadian Tech Job Market Analysis

## Overview
Interactive Power BI dashboard analyzing 50 Canadian tech job postings from April 2025.

## Key Insights
- **Highest Average Salary:** Ontario (~$95K)
- **Most In-Demand Skills:** SQL, Power BI, Python
- **Top Hiring Company:** Shopify
- **Experience Level Salary Gap:** Senior ($120K) vs Entry ($75K)
- **Top Industry:** Finance (12 openings)

## Dashboard Features
- Interactive filters by Province and Experience Level
- Salary analysis by experience and location
- Industry breakdown and company hiring trends
- Responsive visualizations that update based on filters

## Tools Used
- **MySQL:** Data storage and querying
- **Power BI Desktop:** Data visualization and dashboard creation
- **SQL:** Data analysis and aggregation

## Files
- `Dashboard_Screenshot.png` - Screenshot of final dashboard
- `canadian_jobs_data.csv` - Raw dataset (50 jobs)
- `analysis_queries.sql` - SQL queries used for analysis

## How to Use
1. View the dashboard screenshot to see key insights
2. The dashboard is interactive - you can filter by:
   - Province (Alberta, BC, Ontario, Quebec)
   - Experience Level (Entry, Mid, Senior)
3. Each chart updates dynamically based on selected filters

## Key Findings
- Senior positions pay 60% more than entry-level
- Ontario has highest average salary across all levels
- Finance and SaaS industries offer most opportunities
- Most companies have 1-2 openings; few have 3+
